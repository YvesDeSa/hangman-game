# 🎮 Jogo da Forca (Hangman Game)

Um clássico Jogo da Forca (Hangman) desenvolvido como projeto prático para um laboratório de Programação Orientada a Objetos.

## 📝 Sobre o Projeto

Este projeto foi construído para aplicar na prática os conceitos fundamentais de Orientação a Objetos (POO) aprendidos em sala de aula. Através da criação deste jogo, foi possível explorar a modelagem de classes, herança, encapsulamento e o fluxo de regras de negócio em um ambiente interativo.

## 🚀 Tecnologias Utilizadas

* 
**Linguagem:** Java (100% do projeto) 
* 
**Paradigma:** Programação Orientada a Objetos 
* 
**Gerenciamento do Projeto:** Estrutura baseada em IDEs tradicionais do ecossistema Java (contém arquivos `.project` e `.classpath` ).

## ⚙️ Como Executar

Como o projeto possui a estrutura padrão de projetos Java, você pode rodá-lo facilmente usando a sua IDE favorita (como Eclipse, IntelliJ IDEA ou NetBeans):

1. **Clone o repositório:**
```bash
git clone https://github.com/YvesDeSa/hangman-game.git

```


2. **Importe o projeto:**
Abra a sua IDE e selecione a opção para importar um projeto existente a partir da pasta raiz que você acabou de clonar.
3. **Inicie o jogo:**
Navegue até a pasta `src`, localize a classe principal (aquela que possui o método `public static void main(String[] args)`) e execute-a.
